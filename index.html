Academic contribution

Thesis & Research

📖 Undergraduate Thesis

Image Forgery Detection Using Support Vector Machine and Multi-Feature Extraction

📅 Nov 2025 – May 2026 🏫 University of Southern Mindanao 🎓 BS Computer Science

Abstract: This study explores the detection of digitally manipulated images using a Support Vector Machine (SVM) classifier paired with multi-feature extraction techniques. A machine learning pipeline was developed leveraging four image analysis domains — ELA (Error Level Analysis), Histogram, DCT (Discrete Cosine Transform), and DWT (Discrete Wavelet Transform) — to detect and classify forged media with higher precision than single-feature approaches.

🎯
Objective

Develop and evaluate an image forgery detection system using SVM with 4-domain multi-feature extraction

🔬
Methodology

Machine learning pipeline with ELA, Histogram, DCT & DWT feature extraction fed into an SVM classifier

💡
Key Contribution

A robust multi-domain feature fusion approach that outperforms single-feature image forgery detection methods
